DevConnector is a full-stack social network for developers. It lets users build a professional profile, share posts, and connect with other engineers in a clean, modern web experience built on a Next.js frontend and an Express + MongoDB backend.

## What it is
- A community hub for developers to present their skills, experience, and projects.
- A lightweight social feed for sharing updates, ideas, and discussions.
- A profile discovery system to find and connect with other developers.

## Key features
- **Authentication**: JWT-based sign up and login with protected routes.
- **Developer profiles**: Create and edit profiles with skills, bio, location, and social links.
- **Experience & education**: Add, update, and remove professional history.
- **Community posts**: Publish posts, like/unlike, and comment.
- **Profile discovery**: Browse public profiles and view individual detail pages.
- **GitHub integration**: Fetch recent repositories for a user profile.

## Tech stack
- **Frontend**: Next.js (App Router), React, Redux Toolkit, Tailwind CSS.
- **Backend**: Node.js, Express, JWT auth, Express Validator.
- **Database**: MongoDB with Mongoose.
- **Utilities**: Axios, Date-fns, Gravatar.

## Why it stands out
- Clear separation of concerns between API and client UI.
- Real-world user flows (auth, profile management, content creation).
- Solid foundation for scaling into a marketplace or hiring-focused network.

## Ideal use cases
- A portfolio piece for recruiters to evaluate full-stack ability.
- A base product for a developer community or hiring platform.
- A starter for SaaS ideas centered on developer identity and networking.
